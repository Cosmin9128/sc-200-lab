# SC-200 Exam Domains

Overview of exam objectives and study priorities.

---

## Domain 1: Mitigate Threats Using Microsoft Defender XDR (25-30%)

**Topics:**
- Investigate and respond to threats using Microsoft Defender for Endpoint
- Investigate and respond to threats using Microsoft Defender for Office 365
- Investigate and respond to threats using Microsoft Defender for Identity
- Investigate and respond to threats using Microsoft Defender for Cloud Apps
- Manage cross-domain investigations in Microsoft Defender XDR

**Key skills:**
- Navigate the unified security portal (security.microsoft.com)
- Triage alerts and incidents
- Use Advanced Hunting (KQL)
- Configure automated investigation and response
- Manage indicators and exclusions

---

## Domain 2: Mitigate Threats Using Microsoft Sentinel (50-55%)

**Topics:**
- Design and configure a Microsoft Sentinel workspace
- Plan and implement data connectors
- Manage analytics rules
- Configure SOAR (playbooks and automation rules)
- Manage Sentinel incidents
- Use workbooks and hunting
- Manage threat intelligence

**Key skills:**
- KQL queries for detection and hunting
- Create scheduled and NRT analytics rules
- Build Logic Apps playbooks
- Investigate incidents using entity pages and investigation graph
- Use hunting bookmarks and livestream
- Configure data retention and workspace architecture

---

## Domain 3: Mitigate Threats Using Microsoft Defender for Cloud (15-20%)

**Topics:**
- Plan and configure Defender for Cloud
- Manage cloud security posture management (CSPM)
- Configure and manage threat protection

**Key skills:**
- Enable Defender plans for different resource types
- Understand Secure Score and recommendations
- Configure just-in-time VM access
- Manage regulatory compliance dashboard
- Configure cloud workload protection
